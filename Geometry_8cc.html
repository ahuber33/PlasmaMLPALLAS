Detailed Description

Implementation of the Geometry class for the PALLAS project.

This file implements the methods defined in Geometry.hh to construct and retrieve Geant4 geometry volumes used in the PALLAS simulation framework. It provides functions to:

  • Load GDML-defined volumes and assign materials.
  • Create simplified quadrupole magnets as box volumes with vacuum material.
  • Construct diagnostic chambers using subtraction of cylindrical volumes.

The Geometry class acts as a utility for building and managing logical volumes in Geant4 simulations, facilitating the definition of both realistic and simplified detector components.
